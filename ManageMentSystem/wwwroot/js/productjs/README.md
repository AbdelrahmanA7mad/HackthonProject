# Product JavaScript Files

هذا المجلد يحتوي على جميع ملفات JavaScript المتعلقة بصفحات المنتجات، تم فصلها من ملفات العرض لتحسين التنظيم والصيانة.

## الملفات الموجودة:

### 1. `all-products.js`
- **الوصف**: يحتوي على جميع الوظائف الخاصة بصفحة `AllProducts.cshtml`
- **الوظائف الرئيسية**:
  - إدارة المنتجات (إضافة، تعديل، حذف)
  - تصدير إلى Excel
  - البحث بالباركود
  - التحقق من الباركود الفريد
  - إدارة النماذج المنبثقة

### 2. `index.js`
- **الوصف**: يحتوي على الوظائف الخاصة بصفحة `Index.cshtml` (إحصائيات الفئات)
- **الوظائف الرئيسية**:
  - إدارة إحصائيات الفئات
  - طباعة الأسعار
  - إدارة المنتجات في الفئة
  - إدارة المبيعات

### 3. `details.js`
- **الوصف**: يحتوي على الوظائف الخاصة بصفحة `Details.cshtml`
- **الوظائف الرئيسية**:
  - إضافة المنتج للبيع
  - طلب الصيانة
  - إنشاء الأقساط

### 4. `delete.js`
- **الوصف**: يحتوي على الوظائف الخاصة بصفحة `Delete.cshtml`
- **الوظائف الرئيسية**:
  - تأكيد الحذف
  - منع الحذف العرضي
  - إظهار التغذية الراجعة البصرية

### 5. `category-statistics.js`
- **الوصف**: يحتوي على الوظائف الخاصة بصفحة `CategoryStatistics.cshtml`
- **الوظائف الرئيسية**:
  - إدارة إحصائيات الفئات
  - حذف المنتجات والمبيعات
  - إدارة الجداول

## الملفات المطلوبة:

تتطلب هذه الملفات وجود الملفات التالية في المجلد الرئيسي:
- `~/js/price-printing.js` - طباعة الأسعار
- `~/js/barcode.js` - طباعة الباركود
- `~/js/sweetalert-products.js` - التنبيهات

## الاستخدام:

يتم تحميل هذه الملفات في ملفات العرض باستخدام:

```html
@section Scripts {
    <script src="~/js/productjs/all-products.js"></script>
}
```

## المتغيرات العامة:

بعض الملفات تحتاج إلى متغيرات عامة يتم تمريرها من ملفات العرض:

```javascript
// في ملفات العرض
<script>
    window.currentCategoryId = @Model.CategoryId;
    window.currentCategoryName = '@Model.CategoryName';
</script>
```
